# Yousef Tawfiq - Personal Portfolio

Modern, responsive portfolio website with dark mode and data-driven content.

## Structure

- `data/`: All content in JSON files (profile, projects, skills, etc.)
- `css/`: Stylesheet with dark/light theme
- `js/`: Main logic for rendering and interactions
- `public/images/`: Profile image, project thumbnails, certifications, etc.

## How to update

- Edit JSON files in `data/` to change content.
- Replace images in `public/images/` and update paths in JSON.
- No need to touch HTML or CSS for content updates.

## Deployment

Simply upload all files to any static hosting (GitHub Pages, Netlify, Vercel, etc.)

## Custom domain

Configure your DNS and set the custom domain in your hosting provider settings.
